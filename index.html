Open implementation

Federated memory orchestration with NVIDIA FLARE.

The open implementation uses NVFlare's high-level Collab API. Client functions are published with @collab.publish; the server lifecycle is expressed with @collab.main. Sequential training accumulates global memory across domains, followed by frozen parallel evaluation.
